# SpendAI — Portfolio Dashboard Prototype

A working UI prototype for the SpendAI portfolio dashboard, built from a Figma design handoff.

## Stack

- React
- shadcn/ui components
- Tailwind CSS
- Lucide icons

## What this is

This prototype was built to demonstrate rapid design-to-code translation using the same component library the engineering team uses (shadcn/ui). Starting from a rough Figma mockup, the dashboard was matched to the existing design system and converted into production-ready React components.

## Screens

- **Portfolio Dashboard** — Carl's account view with total spend, portfolio companies, and AI-assisted spend analysis

## Design file

Figma: [spendai_1](https://www.figma.com/design/JRCreRQ5lEuly66m9sRlte/spendai_1?node-id=0-1)

```bash
npm install
npm run dev
```
